## 简介

WeAdmin是基于Layui框架设计的Frame型后台管理系统。Layui是由职业前端倾情打造，面向所有层次的前后端开发者，零门槛开箱即用的前端UI解决方案。
Layui发版以来一直受到不少猿的推崇，作为其中的小白，我也一直在推崇和学习中挣扎，WeAdmin无他，也只是小白实操的一个记录。




## 组件说明

### Layui
目前基于Layui 2.3.0版本

### echarts
